# VideoRoom React Application

React приложение для демонстрации Janus VideoRoom plugin.

## Установка

```bash
npm install
```

## Запуск

```bash
npm start
```

Приложение будет доступно по адресу http://localhost:3000

## Настройки

Настройки подключения к Janus серверу находятся в файле `src/settings.js` и соответствуют параметрам из оригинального `settings.js`:

- `server` - адрес Janus сервера
- `iceServers` - STUN/TURN серверы
- `iceTransportPolicy` - политика ICE транспорта

## Использование

1. Нажмите кнопку "Start" для инициализации подключения к Janus
2. Введите имя пользователя и нажмите "Join the room"
3. Разрешите доступ к камере и микрофону
4. Вы увидите свое видео и видео других участников

## Особенности

- Поддержка до 6 активных участников
- Управление битрейтом
- Mute/Unmute аудио
- Unpublish потока
- Отображение информации о битрейте и разрешении для удаленных потоков
